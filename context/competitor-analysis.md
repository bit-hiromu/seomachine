# Competitor Analysis

## 競合メディア分析

### Qiita / Zenn
- **強み**: 記事数が膨大、技術者コミュニティへのリーチが高い
- **弱み**: 個人の深い検証記事は少ない。Claude Code系の日本語詳細ガイドは不足
- **機会**: 実践的なハウツー・比較記事で差別化できる

### 大手テックメディア（ITmedia、Gihyo等）
- **強み**: SEO権威性が高い、PV数が多い
- **弱み**: ニュース速報型が中心。深い実践レビューは少ない
- **機会**: 「実際に試した」視点の詳細記事で差別化できる

### 個人ブログ（note、はてなブログ）
- **強み**: 個人視点で読みやすい
- **弱み**: Claude Code系の日本語ブログはまだ少ない（2026年4月時点）
- **機会**: 早期参入でクラスター全体を押さえられる

## コンテンツギャップ（優先度高）
1. Claude Code 日本語詳細ガイド（ステップバイステップ）
2. Claude Code MCP サーバー実践設定
3. 日本語環境でのAIツール比較検証
4. Claude Code エラー・トラブルシューティング集
5. GitHub Actions × AI自動化 実践例
